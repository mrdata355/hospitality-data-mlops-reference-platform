# MLOps Control Plane Component

**Ownership:** build, test, train, register, deploy, score, monitor, and rollback  
**Deployment targets:** Databricks Workflows, batch tables, FastAPI container, Kubernetes  
**Controls:** CI gates, model acceptance, model aliases, probes, resource limits, monitored outputs

The control plane versions code, contracts, models, metrics, predictions, manifests, and operational documentation together.
